Installation en 2 secondes

Ajoutez l'icône directement sur votre téléphone.

Android

  • 1. Ouvrez le lien dans Chrome.
  • 2. Appuyez sur les 3 points en haut.
  • 3. Cliquez sur "Installer l'application".

iPhone

  • 1. Ouvrez le lien dans Safari.
  • 2. Appuyez sur le bouton Partager.
  • 3. Choisissez "Sur l'écran d'accueil".

Questions Fréquentes

Mes données sont-elles en sécurité ?

Absolument. Vos données sont cryptées et sauvegardées sur des serveurs cloud sécurisés.

Les membres peuvent-ils modifier les comptes ?

Non. Seul le président peut saisir et modifier les données. Les membres utilisent le Mode Vision pour consulter les rapports sans risque d'erreur.

Les rapports sont-ils vraiment en temps réel ?

Oui. Dès que vous enregistrez une cotisation ou un prêt, le bilan est mis à jour instantanément pour tous ceux qui ont le lien de consultation.
